### Tech Stack
<ul>
    <li>Spring Boot</li>
    <li>Hibernate</li>
    <li>JPA Repositories</li>
    <li>PostgreSQL</li>
    <li>Flyway Migrations</li>
</ul>

### Steps
<ol>
    <li>Execute <code>mvn package</code> command</li>
    <li>Then <code>docker-compose build</code> command</li>
    <li>Then <code>docker-compose up</code> command</li>
</ol>
